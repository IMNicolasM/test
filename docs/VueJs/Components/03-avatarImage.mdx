## Usage

1. Include the `avatar-image` component in your template:

   ```jsx
   <avatar-image height="60px" width="60px" src="path/file/image" />
   ```

:::info
The `avatar-image` component is already included in [boot-core-components](/docs/VueJs/modules/QSite/QsiteBoot#core-components) and does not need to be imported separately.
:::
