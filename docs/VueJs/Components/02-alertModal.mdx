## Usage

1. Import and define the `alert-modal` component in your component:

   ```javascript
   import alertModal from '@qsite/_components/master/alertModal'
  ```
2. Use the `alert-modal` component and provide the params prop with the configuration object:

  ```js
  <!-- Alert Modal -->
  <alert-modal :params="componentParams" />
  ```
3. Configure the `componentParams` object with the desired alert information and actions:

  ```js
  componentParams = {
    icon: 'fas fa-cloud-download-alt',
    title: 'Title..',
    message: 'message..', // Load as HTML
    actions: [
      {
        label: 'Ok',
        icon: '',
        color: 'white',
        toUrl: 'https://external-url.com', // Open external URL
        toVueRoute: { name: 'app.home' }, // Redirect to Vue route
        handler: () => {
          // Custom action handler
        },
      },
    ],
  };
  ```
